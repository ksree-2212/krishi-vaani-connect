# 🌱 Crop Recognition and Disease Detection System

## 📌 Project Overview
This project is a web-based application that identifies crop types and detects possible diseases from plant images. Users can upload an image of a crop or leaf, and the system analyzes it to determine the crop type and detect potential diseases.

The platform is designed to help farmers and agricultural workers quickly identify crop issues and take preventive measures to improve productivity and crop health.

---

## 🎯 Features
- 🌾 Crop Identification – Recognizes crop types from uploaded images.
- 🦠 Disease Detection – Detects diseases affecting crops.
- 📷 Image Upload – Allows users to upload plant or leaf images.
- ⚡ Fast Analysis – Provides quick results after processing the image.
- 🖥 Simple Interface – Easy-to-use and responsive design.

---

## 🌾 Farmer-Friendly Interface

This application is specially designed to be accessible for farmers who may not be familiar with English or advanced technology.

### 🌐 Multi-Language Support
Farmers can choose their **preferred language** while using the application.  
Once a language is selected, the interface will display all options and instructions in that language.

### 🔊 Voice Assistance
The system includes a **speaker feature** that reads out the names of buttons and options **before the user selects them**.

This helps farmers who:
- Cannot read English
- Are not comfortable using digital interfaces
- Prefer audio guidance while navigating the application

### 🌱 Crop Pattern Analysis
The platform helps farmers analyze crop patterns by:

- Uploading crop or leaf images
- Identifying crop types
- Detecting possible crop diseases
- Providing insights to help farmers take preventive actions early

## ▶️ Run the Project Locally

Follow these steps:

```sh

git clone <YOUR_GIT_URL>
```
